Vanity BTC Generator

Generiere eine zufälligen Vanity BTC Adresse. BTC Addressen werden mithilfe von base58 erstellt. Aus diesem Grund kann die Addresse nur aus Nummern und Buchstaben ausgenommen l (klein L), O (gross o) und I (gross i) bestehen. Dass generieren von BTC Addressen ist exponentiel Aufänding. Aus diesem Grund sollten maximal 4-5 Buchstaben verwendet werden.
